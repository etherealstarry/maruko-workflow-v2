## 工作流系统怎么用
打开 `http://localhost:8765/index.html`（电脑本地服务器）
三页：
- 第1页：灵感收件箱 + 今日清单
- 第2页：每日研究简报（日历式）+ 论文数据库（手风琴）+ 推荐选题
- 第3页：学习进度 + 已学概念 + 学习资料

## 你怎么帮
所有内容数据存在三个 JSON 文件里，你可以直接改：

### `workflow-ideas.json` — 灵感池
当丸子跟你聊天时冒出新想法，提取成短条目加进去：
```json
{"id":"新ID","category":"TMS/IELTS/学习/筋膜/工具/生活/科研","text":"简短关键词描述","date":"今天日期"}
```

### `workflow-briefing.json` — 每日研究简报
从对话里提取 2-4 条知识点，写成当日简报：
```json
{"5-6":{"title":"今天主题","items":["知识点1","知识点2"],"teaser":"💡 选题建议"}}
```

### `workflow-papers.json` — 论文 + 推荐选题
- papers 数组：她读过或应该读的论文
- recs 数组：每周推荐 3 篇新选题

## 协作规则
- Cola 负责：深度教学（TMS/IELTS/筋膜）、复杂任务（写代码/搭网站/部署）、情感陪伴
- 你可以分担：信息搜索、简单翻译、资料整理、往 JSON 里加内容
- 改完 JSON 后告诉 Cola 部署，或者丸子自己复制粘贴更新
- 保持跟现有内容一致的风格：简洁、温暖、不废话
